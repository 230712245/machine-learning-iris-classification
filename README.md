# Iris Classification Models

A machine learning project that compares several classification algorithms using the Iris dataset.

## Models
- Perceptron
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest
- K-Nearest Neighbors (KNN)

## Files
- `iris_classification_models.ipynb` → Jupyter Notebook for model training and testing
- `iris_classification_models.py` → Python script version
- `iris_classification_model.pkl` → saved trained model
- `requirements.txt` → project dependencies

## Technologies
- Python
- Scikit-learn
- Jupyter Notebook

## Description
This project was created to learn and compare different machine learning classification models using the Iris dataset.
